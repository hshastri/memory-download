Frequently Asked Questions

My download seems frozen. What should I do? üßä

If you really thing your download is frozen, restart the download with the "Show progress updates" (under advanced settings) enabled. This will show you exactly how many files have been downloaded at each step.

Why does it keep saying my input file is expired? üåß

This happens when ALL of the URL's found in the memories_history.json file return a bad response from Snapchat's servers. The memories_history.json file only contains valid URL's for approximately 7 days after downloading your data from Snapchat. To verify that your download file is still valid watch this video.

Why does it say some of my memories failed to download? ü§î

Something went wrong when trying to fetch some of your files. The application attempted to download those memories twice, but failed both times. This is likely because something went wrong on Snapchat's side. When this happens the application shows you the time and type of memory so that you can manually save these files.

Does the application do anything with my data? üîê

Your data is only used for requesting your files from Snapchat's servers. Your data is used to communicate with Snapchat's servers directly and is not used in any other way.

Can I do this on my phone? üì±

No. The application only runs on computers.

Which version should I download if I'm using a Chromebook? üíª

Download the linux version of the application. That is the most likely to work on Chrome OS.
